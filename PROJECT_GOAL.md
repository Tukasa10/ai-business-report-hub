# PROJECT_GOAL

## ゴール

CSVやGoogle Sheetsの業務データを取り込み、集計・可視化・AI分析コメント・改善提案・レポート出力まで自動化するポートフォリオシステムを完成させる。

Phase 1では外部AI APIを呼ばず、サンプルCSVとルールベース分析だけで動くローカルデモを完成させる。

## 完成条件

- サンプルCSVを読み込める
- 売上・注文件数・広告費・CVR・CPA・ROASを集計できる
- 商品別ランキングを出せる
- チャネル別集計を出せる
- 日別推移を可視化できる
- 異常値や改善ポイントを抽出できる
- AI風の分析コメントを生成できる
- 次にやるべき施策を提案できる
- HTMLレポートとして表示できる
- PDF印刷向けレイアウトに対応できる
- CSV出力できる
- README / PORTFOLIO / TEST_CHECKLIST が整っている
- 自動テストが通る

## 非ゴール

- Phase 1で外部AI APIを呼ぶこと
- 実データを使うこと
- APIキーを扱うこと
- 本番運用向けの認証やDBを作ること
- 既存ポートフォリオへ混ぜること
