# レポート設計

## 画面構成

- ヘッダー: サンプル読込、CSV選択、CSV出力、PDF印刷
- ステータス: データ名、期間、品質注意件数
- KPI: 売上、注文、広告費、CVR、CPA、ROAS、返金数、クリック数
- グラフ: 日別売上推移、広告費比較、商品別売上、チャネル別売上
- AI風分析: サマリー、検出ポイント、次にやるべき施策
- 異常値: 広告費ありCVなし、低ROAS、高返金率、日別売上下落、CSV品質注意
- 出力: レポートCSVの対象セクション

## PDF印刷

`@media print` で操作ボタンを非表示にし、A4印刷向けに1列レイアウトへ切り替える。

## デモでの説明順

1. CSVを読み込む
2. KPIを確認する
3. グラフで売上推移と商品別構成を見る
4. 異常値と改善対象を確認する
5. AI風コメントで次の施策を説明する
6. PDF印刷とCSV出力を見せる
