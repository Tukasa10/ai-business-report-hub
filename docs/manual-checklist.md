# 手動確認リスト

Phase 1では外部API、認証、実データ連携は実装しない。以下はユーザーが必要なタイミングで手動確認する。

## 画面・出力

- ブラウザでの最終目視確認
- PDF印刷プレビュー確認
- 印刷後の余白、改ページ、文字切れ確認
- CSVダウンロードファイルをExcel / Google Sheetsで開く確認

## 公開

- GitHubリポジトリ作成
- GitHub Pages公開
- 公開URLでの表示確認
- README / PORTFOLIO の公開前表現確認

## 将来連携

- OpenAI APIキー設定
- Google Sheets連携
- GAS連携
- n8n連携
- Gmail / Slack / LINE通知連携
- 実データ投入
- 本番運用設定

## 注意

- 実データを投入する前に、個人情報、売上機密、顧客情報が含まれていないか確認する。
- APIキーはリポジトリに置かず、環境変数または安全なシークレット管理へ分離する。
- 外部送信を始める場合は、READMEにデータ送信範囲を明記する。
