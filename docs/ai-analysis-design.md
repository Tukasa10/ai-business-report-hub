# AI風分析コメント設計

## Phase 1の方針

外部AI APIは呼ばず、ルールベースでAI風の分析コメントを生成する。

## 入力

- 全体KPI
- 商品別ランキング
- チャネル別集計
- 日別推移
- 前半・後半比較
- 異常値
- CSV品質警告

## 生成するコメント

- エグゼクティブサマリー
- 主要な検出ポイント
- 改善提案
- 次にやるべき施策
- 異常値コメント

## 主なルール

- ROASが低い場合は広告配信・LP・キーワード見直しを提案
- CVRが低い場合は商品ページ・購入導線の改善を提案
- 広告費があるのにCVがない商品は配信抑制候補
- 返金率が高い商品は商品説明・期待値調整・品質確認の候補
- 伸びている商品は予算増額や横展開候補

## 将来拡張

`aiInsightEngine.js` をOpenAI API呼び出しに差し替えることで、同じ集計結果を自然言語分析へ渡せる。
