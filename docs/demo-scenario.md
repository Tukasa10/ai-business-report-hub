# デモシナリオ

## 目的

応募・面談で、CSV集計から改善提案まで一気通貫で対応できることを短時間で説明する。

## シナリオ

1. `npm run dev` でローカル起動
2. ブラウザでサンプルCSVを自動読込
3. KPIで売上、注文、広告費、CVR、CPA、ROASを確認
4. 商品別ランキングで売上上位商品を確認
5. 日別推移で売上の伸び・下落を確認
6. 異常値で「広告費ありCVなし」「返金率高め」を確認
7. AI風分析コメントで改善方針を説明
8. CSV出力でレポート二次利用を説明
9. PDF印刷で提出用レポート化を説明

## 話すポイント

- フロント業務だけでなく、データ集計・可視化・レポート自動化も対応できる
- 外部AI APIなしでも安全にデモできる
- 将来はOpenAI API、Google Sheets、GAS、n8n、Slack/Gmail通知に拡張できる
